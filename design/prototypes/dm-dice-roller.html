Dice Roller
Free roll · Expression · ×N group
▼
—
Roll stage: single-die and multi-group animations, crit/fumble glow, result breakdown chips — identical to DiceRoller.jsx. No changes to markup or animation code.
Apply to…
Apply to...: appears only after a non-d20 roll (damage). Palette-colored pill per party member. Tapping pre-fills the Deal Damage modal — DM still confirms. Disappears when a new roll starts.
d20 mode
Adv/dis strip: identical three-button design and visual states (green/red active). Auto-reset to Normal after every roll. The DM switches creature types constantly — sticky mode causes wrong rolls.
Free Roll
Die picker (d4–d100): same SVG shapes, same selection glow, re-mapped to ember palette variables.
Count
1
Count stepper (1–10): identical circular +/− buttons.
Select a die and count, or type an expression below
+ Modifier
Combo builder: "+ Add to Roll", clear button, pending expression chips, flat modifier input — all identical to player roller.
× 1
×N repeat control: flanks the Roll button. Range 1–8. At ×1 the roll is identical to the player roller. At ×2–8 the roll executes N times independently, showing N labeled result rows (Roll 1, Roll 2 …) with staggered reveal. No sum — each result is discrete. Crit/fumble tagged per row when a d20 is involved.
Expression Roll
Expression input: freeform parser, Enter key trigger — identical to DiceRoller.jsx. The ×N repeat control above also applies to expression rolls.
